# Formatos de Exportación por Caso de Uso

## Análisis de Formatos Óptimos

### 1. Análisis de Variante Individual

| Formato | Uso | Compartir |
|---------|-----|-----------|
| **JSON** | APIs, programadores, integración | Copiar/Pegar |
| **TSV/CSV** | Excel, análisis estadístico | Descargar |
| **PDF** | Reportes clínicos, publicaciones | Email, imprimir |
| **Markdown** | GitHub, documentación | Copiar/Pegar |
| **HTML** | Web sharing con visualización | Link único |
| **VCF anotado** | Pipelines bioinformáticos | Descargar |

### 2. Tracks Genómicos

| Formato | Uso | Compartir |
|---------|-----|-----------|
| **BigWig** | Genome browsers (IGV, UCSC) | Descargar, URL |
| **BedGraph** | Tracks tabulares | Descargar |
| **PNG/SVG** | Imágenes publicación | Copiar, descargar |
| **Interactive HTML** | Visualización web | Link único |

### 3. Tablas de Scores

| Formato | Uso | Compartir |
|---------|-----|-----------|
| **CSV** | Universal | Copiar/Descargar |
| **Excel (.xlsx)** | Análisis empresarial | Email |
| **JSON** | APIs | Copiar |
| **Markdown Table** | GitHub, docs | Copiar/Pegar |

### 4. Batch Results (VCF)

| Formato | Uso | Compartir |
|---------|-----|-----------|
| **VCF anotado** | Pipelines | Descargar |
| **TSV summary** | Análisis rápido | Copiar/Descargar |
| **ZIP bundle** | Todo incluido | Descargar |

### 5. Reportes Clínicos

| Formato | Uso | Compartir |
|---------|-----|-----------|
| **PDF** | Profesional | Email, imprimir |
| **DOCX** | Editable | Email |
| **HTML** | Web | Link único |

---

## Implementación: Componente de Exportación Universal

```typescript
interface ExportOptions {
  formats: ('json' | 'csv' | 'tsv' | 'pdf' | 'markdown' | 'html' | 'vcf' | 'bigwig' | 'png' | 'svg')[];
  data: any;
  filename: string;
}

// Cada resultado tendrá:
// 1. Botón "Copiar" (JSON/Markdown al clipboard)
// 2. Botón "Descargar" (dropdown con formatos)
// 3. Botón "Compartir" (genera link único)
```

## Estructura de Datos Exportable (Ejemplo)

```json
{
  "metadata": {
    "tool": "AlphaGenome Explorer",
    "version": "1.0.0",
    "timestamp": "2026-02-05T12:00:00Z",
    "variant": "chr22:36201698:A>C"
  },
  "summary": {
    "impact": "MODERATE",
    "affected_gene": "APOL4",
    "top_effect": "Expression decrease in Colon"
  },
  "scores": {
    "rna_seq": {
      "gene": "APOL4",
      "tissue": "Colon",
      "raw_score": -0.4523,
      "quantile_score": 0.95,
      "interpretation": "Significant decrease"
    }
  },
  "tracks": {
    "reference": "base64_or_url",
    "alternate": "base64_or_url"
  }
}
```
